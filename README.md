# JHU_CTY_2026_SREH_Prosanta
simulations and lecture notes for the course 'Advanced Topics in Physics: Special Relativity' at Johns Hopkins Center for Talented Youth 2026.


# Gedankenexperiments

Course is designed as a lectures and combinations with the thought experiments and which is termed as 'Gedankenexperiments' (German word for thought experiment). The goal of these experiments to think like a philosopher and a physicist in same time and these thought experiments covers from Plato's world view to Einstein's world view and gives the historical background of how human thought and views undergoes evolution with time.

# Speed of Light
To measure the speed of light an experiment on Focault Rotating Mirror introduced.

# Aether Hypothesis

To get the idea of Fresnel drag the famous Fizeau's experiment introduced here.

# Paradoxes

For reference with the lecture note a visualization of pole barn paradox added here.

# The Optical Illusion

One of the recent descoveries of special relativity 'Terrell-Penrose Effect' added as newer understandings.
